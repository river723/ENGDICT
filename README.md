# 考研英语生词本（离线精简版）

一个专为考研英语设计的生词学习应用，**完全离线、纯本地运行**，无需联网、无需任何 API Key。内置 4801 词的考研增强词库（含词根词缀、例句、易混词、难度标注），开箱即用。

> 本版本由完整版精简而来：移除了全部 AI（DeepSeek）功能、短文生成、考题/完形练习、错题本等需要联网或额外依赖的子系统，只保留纯本地可用的核心学习闭环。

## ✨ 特点

- 🔌 **完全离线**：本地内置词库 + AsyncStorage 持久化，断网即可使用。
- 📚 **内置增强词库**：4801 个考研词汇，含释义、词根词缀、例句、易混词、难度。
- 🧠 **艾宾浩斯复习**：基于遗忘曲线的科学复习计划。
- 🎴 **多种学习模式**：单词卡 / 听写 / 释义选择。
- 🔊 **本地发音**：expo-speech 离线 TTS（Web 端使用浏览器语音合成）。
- 🎨 **主题切换**：浅色 / 深色 / 跟随系统。
- 💾 **数据备份**：导出 / 导入 `.bk` 备份文件。

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn
- Expo CLI

### 安装与运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start
```

在终端按对应键运行：
- iOS：`i`
- Android：`a`
- Web：`w`

无需任何配置，启动后即可直接使用。

### Web 离线打包

```bash
npm run build:web
```

产物位于 `web-build/`，断网后直接打开 `index.html` 即可完全本地运行。

## 📱 核心功能

### 1. 生词管理
- 从本地词库勾选添加（4801 词，已含词根/例句/易混词）
- 手工录入并用本地词库自动补全释义
- 单词详情：释义、词根词缀、易混词、发音

### 2. 学习模式
- 📖 单词卡：翻转卡片背诵
- 🔊 听写：听音拼写
- ✏️ 释义选择：看词选义

### 3. 复习计划
- 艾宾浩斯遗忘曲线自动安排复习
- 每日新词数量可调
- 答错单词自动进入重试队列

### 4. 统计与备份
- 学习记录与准确率统计
- 导出 / 导入 `.bk` 备份文件
- 主题与外观设置

## 🏗️ 项目结构

```
src/
├── components/     # 可复用 UI 组件
├── screens/        # 页面（学习 / 生词本 / 我的）
├── navigation/     # 导航配置
├── services/       # StorageService 等业务服务
├── data/           # 本地词库 worddict.json
├── utils/          # 工具函数（词库查询等）
├── theme/          # 主题
├── providers/      # 主题 Provider
├── types/          # TypeScript 类型定义
└── constants/      # 常量配置
```

## 🔧 技术栈

- **框架**：React Native + Expo SDK 55
- **UI**：React Native Paper
- **导航**：React Navigation（底部 Tab + Stack）
- **存储**：AsyncStorage（本地持久化）
- **发音**：expo-speech
- **语言**：TypeScript

## 🎯 学习计划定制

在「我的 → 学习设置」中可调整：
- 每日新词数量
- 发音开关与自动发音
- 熟词僻义 / 词根词缀显示

复习间隔可在 `src/constants/index.ts` 的 `REVIEW_INTERVALS` 中修改。

## 🐛 常见问题

### Q: 需要联网吗？
A: 不需要。本版本完全离线运行，不依赖任何在线 API。

### Q: 数据如何备份？
A: 在「我的 → 数据管理」中导出 `.bk` 备份文件，需要时再导入恢复。

### Q: 如何调整艾宾浩斯复习间隔？
A: 修改 `src/constants/index.ts` 中的 `REVIEW_INTERVALS` 数组。
